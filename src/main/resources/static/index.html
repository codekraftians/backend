Tech Safe Space API Documentation

Category Endpoints

  • Get All Categories

    Send a GET request to:

    /api/v1/categories

  • Get Category By ID

    Send a GET request to:

    /api/v1/categories/{id}

  • Create a Category

    Send a POST request to:

    /api/v1/categories

    With the following JSON body:

    {
    "type": "Category Name"
}

  • Update a Category

    Send a PUT request to:

    /api/v1/categories/{id}

    With the following JSON body:

    {
    "type": "Updated Category Name"
}

  • Delete a Category

    Send a DELETE request to:

    /api/v1/categories/{id}

Event Endpoints

  • Get All Events

    Send a GET request to:

    /api/v1/events

  • Get Event By ID

    Send a GET request to:

    /api/v1/events/{id}

  • Create an Event

    Send a POST request to:

    /api/v1/events

    With the following JSON body:

    {
    "title": "Event Title",
    "description": "This is the description of the event.",
    "event_date": "2025-05-10",
    "event_time": "15:00",
    "event_image_url": "https://example.com/event-image.jpg",
    "location": "Event Location",
    "max_attendees": 100
}

  • Update an Event

    Send a PUT request to:

    /api/v1/events/{id}

    With the following JSON body:

    {
    "title": "Updated Event Title",
    "description": "Updated description of the event.",
    "event_date": "2025-05-15",
    "event_time": "18:00",
    "event_image_url": "https://example.com/updated-event-image.jpg",
    "location": "Updated Location",
    "max_attendees": 150
}

  • Delete an Event

    Send a DELETE request to:

    /api/v1/events/{id}

User Endpoints

  • Get All Users

    Send a GET request to:

    /api/v1/users

  • Get User By ID

    Send a GET request to:

    /api/v1/users/{id}

  • Create a User

    Send a POST request to:

    /api/v1/users

    With the following JSON body:

    {
    "name": "User Name",
    "email": "user@example.com",
    "password": "Password123!",
    "user_image_url": "https://example.com/user-image.jpg"
}

  • Update a User

    Send a PUT request to:

    /api/v1/users/{id}

    With the following JSON body:

    {
    "name": "Updated User Name",
    "email": "updateduser@example.com",
    "password": "UpdatedPassword123!",
    "user_image_url": "https://example.com/updated-user-image.jpg"
}

  • Delete a User

    Send a DELETE request to:

    /api/v1/users/{id}
